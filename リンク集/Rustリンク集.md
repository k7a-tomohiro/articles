
Rustで開発するときや学習する時に参考にできるリンク集です。
随時更新。

# 公式サイト

- **[Rust Programming Language](https://rust-lang.org/)**: Rustの公式サイト。
- **[rust-lang/rust](https://github.com/rust-lang/rust)**: Rust本体のリポジトリ。
- **[Rust Documentation](https://doc.rust-lang.org/stable/)**: →[日本語版](https://doc.rust-jp.rs/)
  - **[The Rust Programming Language](https://doc.rust-lang.org/book/)**: →[日本語版](https://doc.rust-jp.rs/book-ja/)
  - **[The Rust Reference](https://doc.rust-lang.org/reference/)**: Rustの言語仕様が説明されている。
  - **[The Rustonomicon](https://doc.rust-lang.org/stable/nomicon/index.html)**: Rust裏本と呼ばれているもの。
- **[crates.io](https://crates.io/)**: クレート検索サイト。
- **[docs.rs RustDocs](https://docs.rs/)**: RustDoc形式のドキュメント検索サイト。
